# Building a Personal AI OS on a Knowledge Graph

A single, self-contained, interactive page documenting a personal "AI operating system": skills (apps) sitting on top of a markdown knowledge graph in Obsidian — a medallion (bronze/silver/gold) data architecture, a 7-stage skill chassis, an explicit model-vs-code boundary, and a deterministic hook harness.

**▶ Live:** https://ahasham.github.io/ai-os-showcase/

`index.html` is fully self-contained (inline CSS/JS/SVG, no external requests) — it renders offline if you open the file directly.

Names are pseudonyms; the citizenship-form example is schema-only; secrets live in 1Password.
